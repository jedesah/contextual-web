    The Contextual type is a representation of the known compile-time information about an interpolated string. Most importantly, this includes the literal parts of the interpolated string; the constant parts which surround the variables parts that are substituted into it. The Contextual type also provides details about these holes, specifically the possible set of contexts in which the substituted value may be interpreted.
